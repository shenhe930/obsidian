# Homogenous Coordinates 

**Add a third coordinate (w-coordinate)**
- 2D point = $(x, y, 1)^T$
- 2D vector = $(x,y,0)^T$

**Valid operation if w-coordinate of result is 1 or 0**
 - vector + vector = vector
 - point - point = vector
 - point + vector = point
